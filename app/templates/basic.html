DOIFs - Analisador de portarias das IFs

Experimente
Sobre

é uma plataforma de recuparação de informações das portarias das IFs (Instituições Federais de ensino).
Extração

A fim de análisar o comportamento de atos de pessoal publicados no DOU referentes as IFs o sistema extrair os dados através de Web scraping/Web crawler, sem uso de API
Filtros

O sistema realiza filtragem nos dados obtidos para posterior análise. Inicialmente, os atos recuperados sã de NOMEAÇÔES e EXONERAÇÔES.
Limitações

A recuperação de dados por meio de Web scraping/Web crawler ocasiona algumas limitações como um tempo de processamento demorado.
